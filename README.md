# MatteTrening

Et sanntids matematikkverktøy for klasserom.

## Krav

- [Node.js](https://nodejs.org/) v18 eller nyere

## Kom i gang

### 1. Installer og start (første gang)

```powershell
.\start.ps1
```

Dette installerer alle avhengigheter og starter både server og klient.

### 2. Etterfølgende oppstart

Server:
```powershell
cd server
npm run dev
```

Klient (i nytt terminalvindu):
```powershell
cd client
npm run dev
```

Åpne **http://localhost:5173** i nettleseren.

## Bruk

**Lærer:**
1. Klikk "Jeg er lærer"
2. Velg tema og sett eventuelt mål/nedtelling
3. Del koden med elevene
4. Klikk "Start økt" når alle er inne
5. Se elevenes progresjon i sanntid
6. Etter avslutning: gjennomgå feil på tavla

**Elev:**
1. Klikk "Jeg er elev"
2. Skriv inn 6-sifret kode
3. Vent på at læreren starter
4. Svar på oppgavene!
